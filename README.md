## 📌 About the Project

**Apple Replica** is a frontend + backend clone of the Apple Store website, built as a field project for our 4th semester Full Stack Development course. The goal was to replicate the look, feel, and functionality of Apple's official web store using modern web technologies.

> 🚀 Deployed live on **Vercel**: [apple-replica-five.vercel.app](https://apple-replica-five.vercel.app)

---

## 🛠️ Tech Stack

| Layer      | Technology                              |
|------------|------------------------------------------|
| Frontend   | React 18, React Router DOM v6, Vite     |
| Styling    | CSS (31%)                               |
| Icons      | Lucide React                            |
| Backend    | Node.js (Express)                       |
| Deployment | Vercel                                  |

## ⚡ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation & Running Locally

```bash
# Clone the repository
git clone https://github.com/084divyanshuraj/Apple_Replica.git
cd Apple_Replica

# Install dependencies
npm install

# Start the development server
npm run dev
```

> **Windows users:** You can also just double-click `Start_Apple_Replica.bat` to launch the project quickly.

### Build for Production

```bash
npm run build
npm run preview
```
