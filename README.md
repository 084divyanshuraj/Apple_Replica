## 📌 About the Project

**Apple Replica** is a frontend + backend clone of the Apple Store website, built as a field project for our 4th semester Full Stack Development course. The goal was to replicate the look, feel, and functionality of Apple's official web store using modern web technologies.

> 🚀 Deployed live on **Vercel**: [apple-replica-five.vercel.app](https://apple-replica-five.vercel.app)

---

## 🛠️ Tech Stack

| Layer      | Technology                              |
|------------|------------------------------------------|
| Frontend   | React 18, React Router DOM v6, Vite     |
| Styling    | CSS (31%)                               |
| Icons      | Lucide React                            |
| Backend    | Node.js (Express)                       |
| Deployment | Vercel                                  |
